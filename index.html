#Que es Html PRO?

  • Generador de codigo html profesional con una cantidad mayor de meta etiquetas que incluyen autor, description, keywords, theme-color, twitter-card etc.

#como?

  • Usando los snippets `html-basic` o `html-pro`, se genera c칩digo HTML plano, usando `html-basic.` se genera codigo html con sus archivos CSS y javaScript pero con `html-pro/` se genera una carpeta para los estilos y archivos javaScript.

Snippets

html-basic

Estructura de codigo html con las meta etiquetas necesarias para un proyecto.

html-basic.

Estructura de codigo html con las meta etiquetas necesarias para un proyecto y sus respectivos archivos css, javaScript.

html-pro

Estructura de codigo html profesional, incluye meta etiquetas como style, twitter-card, apple-touch-icon al instante.

html-pro/

Crea una carpeta style y Script incluyendo todo el codigo del snippets html-pro
